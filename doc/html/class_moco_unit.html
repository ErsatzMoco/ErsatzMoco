Abstract base class for any ErsatzMoco construction.

Classes derived from MocoUnit represent the physical box etc. that makes use of lower level components it consists of. One single MocoUnit child must be instantiated in the .ino file (which defines setup and loop function).

See also
MocoUnitType, Beamerbox, Boat, KSRemote, Locomotive, Remote
