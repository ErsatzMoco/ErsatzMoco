◆ AdafruitOLED()

AdafruitOLED::AdafruitOLED ( )

Wrapper for displays using the SSD 1331 driver chip. Connects to the corresponding library by Adafruit.

Wrappers of base type Display implement the complete set or a subset of standard methods for constructing graphical elements. Pins for talking to driver chip (SPI) must be set up in header file AdafruitOLED.hpp!
